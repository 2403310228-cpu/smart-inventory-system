# Smart Inventory System

Desktop-based Inventory and Sales Management System developed using Java Swing and MySQL.

## Features
- Inventory Management
- Supplier Management
- Sales Transactions
- Dashboard
- Sales Reports
- Receipt Printing (JasperReports)

## Technologies
- Java
- Java Swing
- MySQL
- Maven
- JasperReports
- JFreeChart

## How to Run
1. Import the MySQL database.
2. Open the project in NetBeans.
3. Configure the database connection.
4. Run the application.
